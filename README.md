# javascript-challenge

Code to create a dynamic table based on information on UFO sightings. Website uses JavaScript, HTML, CSS, and D3.js to create a table that can be filtered by date and stae by users. Available date range is 1/1/2010 to 1/13/2010. Pop-up alert included if date entered is out of the available range. States listed in alphabetical order in dropdown.

Data columns displayed:

-Date

-City

-State

-Country

-Shape

-Duration

-Comments

Direct link to website: https://vl14.github.io/javascript-challenge/
